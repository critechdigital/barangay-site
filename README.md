## Tech Stack

- Ruby on Rails 8.1
- SQLite (development) / PostgreSQL (production)
- Tailwind CSS v4
- Hotwire (Turbo + Stimulus)
- Minitest (60 tests)

## Getting Started

```bash
# Install dependencies
bundle install

# Setup database
rails db:migrate

# Run tests
rails test

# Start server
rails server
```

## Development

```bash
bin/dev  # Starts Rails + Tailwind watcher
```
